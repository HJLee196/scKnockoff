# README

# scKnockoff

An R package for differential analysis of single-cell RNA sequencing (scRNA-seq) data using knockoff-based variable selection, supporting missing data and employing a matrix decomposition-based knockoff constructor for efficient inference in high-dimensional datasets.

## Installation

Install from GitHub:
```r
devtools::install_github("HJLee196/scKnockoff")
```
